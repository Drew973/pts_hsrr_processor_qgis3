Readings Table

Table containing readings.

  • run:which run the readings are in. Must be in run_info.
  • t:timestamp
  • f_line: which line of file data is from
  • raw_ch:chainage of the reading
  • rl:reading
  • vect:linestring geometry from start_point to end_point
  • s_ch: start chainage of reading(km)
  • e_ch: end chainage of reading(km)

s_ch and e_ch are calculated from f_line to ensure uniqueness (raw_ch sometimes decreases).
