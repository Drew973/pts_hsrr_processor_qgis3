The "prepare new database" button creates the required schemas,tables,views and functions

These will be in the hsrr,routes and topology schemas

The postgis and topology extensions are required.

Values will need to be inserted into the network table before fitting.

Topology based autofit requires topology tables.The hsrr.recalc_topology(tol float =5) sets this up. It should be called after changes to hsrr.network.
