Network table

Table containing road network:

  • sec : section label. Unique primary key.
  • geom : linestring geometry.
  • meas_len : 'official' length of section. Used to calculate section chainage and in resizing.
  • has_forward: Section can be surveyed in forward direction. Used in autofitting.
  • has_reversed: Section can be surveyed in reverse direction. Used in autofitting.
